### About

Converting data from MSCONS (EDIFACT) format to CSV.

### Dependencies

* Python 2.7

### Usage
```
python msconsconverter.py
```

### Materials MSCONS (EDIFACT)

* [EDIFACT - German](https://de.wikipedia.org/wiki/EDIFACT)
